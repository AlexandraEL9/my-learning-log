HTML Introduction

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It defines the elements of a webpage, such as headings, paragraphs, images, links, and tables, using tags. HTML allows browsers to interpret and display content in a structured way, forming the foundation of every website. It works alongside CSS (for styling) and JavaScript (for interactivity).

The Evolution of HTML & the Web

Origins of Markup Languages (1960s - 1980s)

  • IBM created GML (General Markup Language) for machine-readable document sharing.
  • GML evolved into SGML (Standard Generalized Markup Language), which structured documents.
  • SGML lacked hypertext (no embedded links between documents).

CERN & The Need for a Better System (Late 1980s)

  • CERN researchers needed a better way to share and cross-reference research documents.
  • Tim Berners-Lee proposed linking research documents instead of just downloading them.

The Birth of Hypertext & The Web (1991)

  • Berners-Lee developed a system to link documents via hypertext.
  • He created:
    • HTTP (Hypertext Transfer Protocol) – rules for network communication.
    • HTML (Hypertext Markup Language) – based on SGML but added hyperlinks.
    • The first web browser – to display linked documents.
    • URL (Uniform Resource Locator) – a way to locate documents.
    • Web server & client model – servers store and deliver documents upon request.

Growth of the Web (Early 1990s - Mid 1990s)

  • Others expanded HTML by adding new software and structural tags.
  • Universities, hobbyists, and media companies started creating websites.
  • Images were integrated, making the web more dynamic.
  • Within 5 years, the web exploded in size & complexity.

The Anatomy of a Web Page

HTML & Elements

  • HTML structures content using **elements**, which act as **instructions** for browsers.
  • Files are saved with a **.html** extension so browsers recognize them.
  • Browsers interpret **elements** to display content as intended (e.g., headings, paragraphs, bold text, lists, tables).
  • Despite modern advancements, web pages from **1991 still work in today’s browsers**.

HTML Elements & Tags

  • An **element** consists of:
    • An **opening tag** (e.g., <h1>).
    • The **content** (e.g., "This is a heading").
    • A **closing tag** (e.g., </h1>).
  • Tags tell the browser how to format content (e.g., headings appear bold and large).
  • The browser reads from **left to right** and stops formatting when it reaches a **closing tag**.

Nested Elements & Parent-Child Relationships

  • Elements can **contain other elements** (e.g., a paragraph can contain an emphasis tag).
  • The **parent-child relationship** creates a **tree structure** (Document Object Model, or **DOM**).
  • Browsers treat page elements as **objects** with properties (font, color, size, etc.).
  • Technologies like **CSS & JavaScript** can modify these objects to create interactive pages.

Basic Structure of an HTML Document

  • A minimal HTML page requires three core elements:
    • <html> – The **root element**, parent of all others.
    • <head> – Contains **meta information**, page title, and resource links.
    • <body> – Contains the **visible content** of the webpage.
  • External resources (e.g., fonts) are linked in the **head** and automatically loaded by the browser.

HTML Structure & Web Development Basics

HTML, CSS & JavaScript: The Core of Web Development

  • HTML (Hypertext Markup Language) – Defines the structure of a webpage using elements and tags.
  • CSS (Cascading Stylesheets) – Styles the webpage, controlling fonts, colors, layouts, and animations.
  • JavaScript – Adds interactivity, enabling dynamic updates, animations, and user interactions.

Basic HTML Structure

  • HTML files must be saved with a .html extension.
  • The structure includes:
    • <!DOCTYPE html> – Declares the document as HTML5.
    • <html> – The root element containing all page content.
    • <head> – Stores metadata, page title, and linked resources.
    • <body> – Contains visible content such as text, images, and links.

Understanding HTML Elements

  • HTML is made up of **elements and attributes**.
  • Elements use **tags** (e.g., <p>This is a paragraph</p>).
  • Each HTML page follows the same **basic structure**.
  • Elements **instruct the browser** on how to display content.

HTML files can be edited with any text editor and viewed in any browser.
